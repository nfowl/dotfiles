# Dotfiles
My (Currently Barebones) Dotfiles for WSL/Arch Linux Managed via [GNU stow](https://www.gnu.org/software/stow/)

## Current toolset
* vim **Basic support**
  * Expand this to support better tooling
* zsh
* mpd
* ncmpcpp
* kitty (Arch Linux only)
* git
* starship

## Supported tools
- **ZSH**
  - Antibody plugin manager
  - Kitty (Optional)
  - Kubectl (Optional)
  - golang (Optional)
  - Rust (Optional)
